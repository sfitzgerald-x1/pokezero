# Endgame Scenario Seeds

These are source-composed, fully revealed Gen 3 randbats scenarios for the local
`pokezero-scenarios` studio. They are a materialization and regression suite, not an
in-distribution strength benchmark: every file is `set-valid` and `state-consistent`, but
remains `unverified` until a human review or a future forced-line proof artifact supplies a
concrete winning claim.

The suite deliberately spans distinct root-decision shapes: priority, low-HP ability thresholds,
recoil preservation, finite PP, recharge commitment, trapping, recovery pressure, sleep timing,
item thresholds, and an active-slot/party-preservation case. The regression test loads every JSON
file and materializes it through Showdown.
